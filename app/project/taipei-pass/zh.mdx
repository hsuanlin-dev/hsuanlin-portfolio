# 台北通 — 城市級數位生態系的設計與實作橋樑

我以**「專案設計師」的身分主導前端實作**，負責台北市政府（台北通）數位生態系的建設。配合市府「無紙化與數位轉型」政策，我的核心任務是將繁瑣的行政程序轉化為直覺、高效能的數位服務。

> **Project Link:** [https://id.taipei/tpcd/](https://id.taipei/tpcd/) ↗

## 專案背景

台北通是一套服務**百萬市民的城市級數位基礎建設**，透過統一的數位入口整合台北市各項政府服務。平台橫跨多樣化技術棧，涵蓋 React 應用與 Bootstrap 入口網站，在複雜的異質架構下運行。

主要負責範疇：

- 帶領 5 人前端開發團隊
- 建立 Figma 視覺稿與代碼實作之間的嚴格審核機制
- 主導全平台無障礙標準（A11y AA）實作
- 定義跨技術棧的組件化開發規範，確保品質一致

## 核心挑戰

**消除實作斷層：** 靜態設計規範在開發過程中往往會失真。我的挑戰是擔任**「最終轉譯者 (The Final Translator)」**，確保設計願景能精準地轉化為程式碼。

**全民無障礙要求：** 在政府級入口網，「確保每個人都能使用」是最高原則，這需要極度嚴謹的技術手段來落實無障礙標準。

## 戰略解決方案

我建立了一套品質優先的工作流，帶領開發團隊達成以下標竿：

- **像素級還原的承諾：** 我發揮設計背景優勢，在 Figma 視覺稿與代碼實作間建立嚴格審核機制，確保產出達成 **100% 的設計精準度**。
- **無障礙 AA 級深耕：** 我主導落實了從語意化標籤到鍵盤操作邏輯的完整實作，協助系統通過國家級無障礙 2.1 AA 標章認證。
- **異質技術棧治理：** 針對 React 與 Bootstrap 等不同技術需求的專案，定義了一套跨棧的組件化開發標準，確保團隊在併行處理多項子專案時，依然能維持視覺高度一致與高品質的代碼產出。

## 核心貢獻

- **設計與工程的跨領域協作：** 與市府設計師密切協作，從開發角度提供技術建議，確保設計系統能被最有效地轉化為可擴展的組件。
- **政務流程的 UX 優化：** 將複雜的行政邏輯優化為直覺的前端介面，顯著降低了市民與公務人員的使用門檻。
- **品質把關者與技術導師：** 擔任 UI/UX 實作的技術主導，指導團隊落實現代前端最佳實踐，並堅持對視覺細節的極致追求。

## 最終成果

透過這個專案，我實踐了：

- **可擴充的數位基礎建設：** 成功建立了一套可長期維護的數位生態系，為台北市的無紙化革新打下堅實基礎。
- **技術與美學的深度整合：** 實踐了「工程穩健性」與「視覺優雅」的高度統一，為大規模政府平台樹立了全新的品質標竿。
- **架構適應力的展現：** 在異質技術棧並存的環境中，不是「工具的使用者」，而是能制定跨棧治理規範的 **Architect**。
