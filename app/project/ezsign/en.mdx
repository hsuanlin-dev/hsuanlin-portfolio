## Overview

I owned the frontend visual implementation for EZSIGN, a B2B digital signature platform — including a multi-step Wizard workflow and a landing page for the onboarding flow. The underlying business logic and functionality were handled by other engineers on the team.

## Challenge

The signing process couldn't be collapsed into a single form: documents needed uploading, one or more signers had to be configured in a specific order, and notification preferences had to be set. Structuring all of that into a guided flow, without overwhelming the user with one dense form, was the core challenge. The existing tech stack was also a constraint rather than a choice — the project already ran on Bootstrap — and the visual presentation needed a full re-theme to feel trustworthy for a legal process.

## What I Did

- Built a multi-step Wizard — basic information, document upload, then signer and notification setup — guiding users through the signing flow without overwhelming them
- Re-themed the platform's visual presentation entirely, reworking the color system within the existing Bootstrap setup
- Designed and implemented the landing page so visitors could immediately understand what the service does and why to choose Tradevan

## Outcome

The Wizard flow and re-themed interface shipped as part of EZSIGN's B2B onboarding experience. I don't have post-launch metrics to point to, but the goal was clear: make a legally dense, multi-step process feel guided and low-friction at the UI level.
