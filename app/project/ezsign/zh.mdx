# EZSIGN — 提升 B2B 轉化率的策略性 UI 與信任驅動體驗

我主導了 **EZSIGN** B2B 電子簽章平台的 UI/UX 策略與前端實作。以 **Bootstrap 5** 作為核心響應式引擎，我致力於將複雜的法律流程轉化為「人性化」且高轉化率的數位旅程。

## 專案背景

EZSIGN 是一套專為企業用戶設計的 **B2B 電子簽章平台**，服務於需要處理關鍵法律交易的企業場景。專案要求在嚴格的法律合規、視覺信任感，以及跨各種企業軟硬體環境的無縫相容性之間，取得精準的平衡。

主要負責範疇：

- 主導完整的 UI/UX 策略規劃與前端實作
- 透過深度客製化 Bootstrap 5 SCSS，重新架構整套 UI 系統
- 設計引導式多步驟精靈流程（Wizard UX），降低使用者操作摩擦
- 建立模組化組件庫，支撐 B2B 產品線的長期可擴充性

## 核心挑戰

**「法律焦慮」的門檻：** 傳統 B2B 簽署涉及密集的表單與艱澀術語，容易在關鍵交易過程中導致使用者產生挫折感。

**穩定性的工程挑戰：** 挑戰在於如何交付具備高度信任感的客製化視覺，同時確保在各種企業軟硬體環境下達成**極致的跨瀏覽器相容性**與穩定性。

## 戰略解決方案

我採取了**「以人為本的 B2B 策略」**，並透過深度的架構客製化來實現：

- **客製化 Bootstrap 架構：** 我並非採用通用外觀，而是透過**深度複寫 Bootstrap 的 SCSS 變數**重新架構了 UI 系統。這在維持框架企業級穩定性的同時，打造出專業的「法律科技（LegalTech）」美學。
- **引導式精靈流程 (Wizard UX)：** 利用 **Bootstrap 的網格與模態系統**，將複雜程序拆解為多步驟介面。這種分段處理顯著降低了認知負荷，並引導使用者清晰地完成關鍵任務。
- **工具優先的客製化：** 運用 Bootstrap 的 Utility-first 原則確保快速的 UI 迭代，同時維持從設計到程式碼的**像素級還原度（Pixel-perfect）**。

## 核心貢獻

- **可擴充的 UI 基礎：** 在 **Bootstrap** 之上建立了模組化組件庫，確保整個 B2B 產品線具備低門檻的維護性與高度一致性。
- **戰略性 UX 把關：** 橋接艱澀的法律需求與以人為本的設計，優化流程以顯著降低使用者的流失率。
- **高效能實作：** 優化前端架構，確保基於 **Bootstrap 的介面**在處理繁重法律文件時，依然能維持輕量化與即時響應。

## 最終成果

透過這個專案，我實現了：

- **實證的商業轉化：** 將艱澀的法律流程轉化為直覺的旅程，證明了**在穩健框架上進行設計驅動開發**，能直接提升商業營運效率。
- **企業級可靠性：** 交付了一個極其穩定且具備擴充性的平台，為高信任感的 B2B 數位產品樹立了品質標竿。
- **架構掌控力的展現：** 證明了對框架進行深度客製化——而非替換框架——往往是在企業產品開發中同時達成速度與品質的最佳戰略路徑。
