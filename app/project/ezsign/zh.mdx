# EZSIGN — 提升 B2B 轉化率的策略性 UI 與信任驅動體驗

![image](/ezsign.png)
> **Project Link:** [https://ezsign.tradevan.com.tw/APESP/](https://ezsign.tradevan.com.tw/APESP/) ↗

## 專案背景

我負責 EZSIGN(B2B 電子簽章平台)的前端視覺實作——包括導入流程的多步驟 Wizard 流程與 landing page。底層的業務邏輯與功能則交由團隊裡其他工程師處理。

## 核心挑戰

簽署流程沒辦法塞進單一表單:文件需要上傳,一個或多個簽署人要依序設定,通知偏好也要配置。把這些步驟組織成一個循序漸進的流程,不讓使用者一次面對過於複雜的表單,是這個案子的核心挑戰。技術選型上也不是自由發揮的空間——案子本來就用 Bootstrap——視覺呈現則需要整套重新換色系,才能讓這個法律性質濃厚的流程感覺值得信任。

## 實際做的事

- 打造多步驟 Wizard 流程——基本資料、文件上傳,再到簽署人與通知設定——引導使用者完成設定,不會一次被太多資訊淹沒
- 在既有的 Bootstrap 架構下,整套重新設計色彩系統與視覺呈現
- 設計並實作平台的 landing page,讓訪客一進站就能看懂這項服務在做什麼、為什麼該選擇關貿網路

## 成果

Wizard 流程跟重新換色後的介面已經上線,成為 EZSIGN B2B 導入體驗的一部分。我手上沒有上線後的數據可以佐證效果,但目標很明確:讓一個法律意味濃厚、步驟繁多的流程,在 UI 層面感覺更有引導感、更低摩擦。
