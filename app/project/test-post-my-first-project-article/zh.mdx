# 我的第一個專案文章

這篇文章介紹我的第一個專案，以及我在開發過程中的一些設計思考。

## 專案介紹

這個專案是一個個人作品集網站，使用 **Next.js** 和 **TypeScript** 建構。

主要功能包含：

- 部落格文章系統
- 專案展示頁
- 中英文語言切換
- 深色模式

## 技術使用

這個專案使用了以下技術：

- Next.js App Router
- TypeScript
- TailwindCSS
- MDX

## 開發心得

在這個專案中，我學到了如何：

- 建立 MDX 文章系統
- 實作多語系網站
- 組織 React component 架構

未來我也會持續優化這個網站。
