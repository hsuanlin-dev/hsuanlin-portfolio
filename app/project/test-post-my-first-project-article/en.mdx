# My First Project Article

This article introduces my first project and the design decisions I made during development.

## Project Overview

This project is a personal portfolio website built with **Next.js** and **TypeScript**.

Main features include:

- Blog article system
- Project showcase pages
- Language switching (English / Chinese)
- Dark mode support

## Tech Stack

The technologies used in this project include:

- Next.js App Router
- TypeScript
- TailwindCSS
- MDX

## What I Learned

Through this project, I learned how to:

- Build an MDX-based blog system
- Implement a multilingual website
- Structure React components effectively

I will continue improving this website in the future.
